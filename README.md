# theatre-api 🎭

## 📝 Description
**theatre-api** is a modern RESTful API service designed to fully automate theater operations. The primary goal of the project is to provide theater-goers with a convenient way to browse the current repertoire, view performance schedules, and book specific seats in real-time, eliminating the need to visit the box office physically.

For theater administrators, the system provides a robust tool for managing performance schedules, hall configurations, cast members, and genre diversity. The service ensures strict ticket validation, making it impossible to double-book the same seat for the same performance.

---

## 🛠 Technologies
- **Python 3.10+**
- **Django**
- **Django REST Framework (DRF)**
- **PostgreSQL** (or SQLite for local development)
- **JWT Authentication**
- **Swagger/ReDoc Documentation**
